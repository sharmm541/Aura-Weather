## üåê API Integration

This app uses the [OpenWeatherMap API](https://openweathermap.org/api) to fetch weather data:

- **Current Weather API**: `https://api.openweathermap.org/data/2.5/weather`
- **5-Day Forecast API**: `https://api.openweathermap.org/data/2.5/forecast`
- **Geolocation API**: Coordinate-based weather lookup

### API Features Used:
- Real-time temperature, humidity, pressure
- Wind speed and direction
- Weather conditions and descriptions
- 5-day weather forecast
- Multi-language support (Russian/English)
